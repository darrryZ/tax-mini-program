## UI设计特点

### 配色方案
- **主色调**: #07c160 (微信绿)
- **渐变色**:
  - 个税计算器: 紫色渐变 (#667eea → #764ba2)
  - 五险一金: 粉红渐变 (#f093fb → #f5576c)
  - 企业成本: 蓝色渐变 (#4facfe → #00f2fe)
  - 贷款计算: 绿色渐变 (#43e97b → #38f9d7)

### 组件样式
- **卡片**: 白色背景，圆角20rpx，阴影效果
- **按钮**: 圆角10rpx，主按钮绿色，次按钮灰色
- **输入框**: 边框1rpx，圆角10rpx，高度80rpx
- **结果展示**: 渐变色头部 + 白色内容区
- **数据项**: 左右布局，标签居左，数值居右

### 交互反馈
- 点击卡片: 背景色变化
- 计算按钮: 绿色主题色
- 重置按钮: 灰色次要色
- 输入验证: Toast提示
- 结果展示: 渐变色高亮

```

   ↓
[输入金额]
   ↓
[检查有效性]
   ↓
[调用utils中的计算函数]
   ↓
[返回计算结果对象]
   ↓
[更新页面data]
   ↓
[渲染结果到页面]
```

## 使用示例

### 示例1: 计算月薪10000的个税
```
输入：
- 月工资: 10000
- 专项附加扣除: 1000
- 计算方式: 月度计算

输出：
- 税前收入: 10000.00
- 五险一金: 2250.00
- 应纳税所得额: 21000.00
- 适用税率: 3%
- 应缴个税: 52.50
- 税后收入: 7697.50 ✓
```

### 示例2: 计算五险一金
```
输入：
- 工资基数: 10000

输出：
- 个人缴纳总计: 2250.00
- 单位缴纳总计: 3980.00
- 合计: 6230.00
```

### 示例3: 计算企业成本
```
输入：
- 员工月工资: 10000
- 员工数量: 5
- 其他月度成本: 1000

输出：
- 月度总成本: 70900.00
- 年度总成本: 850800.00
```

### 示例4: 计算贷款
```
输入：
- 贷款金额: 1000000
- 年利率: 4.9
- 贷款年限: 30
- 还款方式: 等额本息

输出：
- 月还款额: 5307.27
- 支付总额: 1910616.19
- 支付利息: 910616.19
```

## 总结

本应用提供了完整的税务计算功能，界面简洁美观，操作流畅，计算准确。适合个人用户进行税收规划，也适合企业HR进行成本核算。
