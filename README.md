# 税务计算微信小程序

一个功能全面的税务计算微信小程序，提供个税、五险一金、企业成本和贷款计算功能。

## 功能特性

### 1. 个税计算器 (💰)
- 支持月度和年度个税计算
- 基于2019年最新个税法
- 自动计算五险一金扣除
- 支持专项附加扣除
- 显示适用税率和应纳税额
- 实时计算税后收入

### 2. 五险一金计算器 (🏥)
- 计算养老保险、医疗保险、失业保险、工伤保险、生育保险
- 计算住房公积金
- 分别显示个人和单位缴纳金额
- 详细的缴费明细
- 支持自定义缴费比例

### 3. 企业成本计算器 (🏢)
- 计算企业雇佣员工的总成本
- 包含工资和单位承担的五险一金
- 支持多员工计算
- 月度和年度成本分析
- 支持其他成本输入

### 4. 贷款计算器 (🏠)
- 支持等额本息和等额本金两种还款方式
- 详细的还款计划表
- 计算总利息和每月还款额
- 可查看完整还款明细
- 支持不同贷款年限

## 技术特点

- 基于微信小程序框架开发
- 模块化设计，易于维护和扩展
- 所有计算逻辑独立封装在utils目录
- 响应式UI设计
- 符合中国税法和社保政策

## 使用说明

1. 在微信开发者工具中打开项目
2. 选择所需的计算器功能
3. 输入相关参数
4. 点击"计算"按钮查看结果

## 注意事项

- 计算结果仅供参考，具体以当地政策和官方规定为准
- 五险一金缴费比例可能因地区而异
- 个税计算基于2019年最新个税法
