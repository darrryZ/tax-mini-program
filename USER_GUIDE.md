# 税务计算工具使用指南

## 快速开始

本应用是一个微信小程序，提供四大核心功能：个税计算、五险一金计算、企业成本计算和贷款计算。

## 功能详解

### 1. 个税计算器

**使用步骤：**
1. 进入"个税计算器"页面
2. 输入月工资或年收入
3. （可选）输入专项附加扣除金额（如子女教育、住房贷款利息等）
4. 选择计算方式：月度计算或年度计算
5. 点击"计算"按钮

**输出结果：**
- 税前收入
- 五险一金扣除额（自动计算）
- 专项附加扣除
- 应纳税所得额
- 适用税率
- 应缴个税
- 税后收入

**示例：**
- 月工资：10,000元
- 专项附加扣除：1,000元/月
- 计算结果：税后收入约7,697.50元

### 2. 五险一金计算器

**使用步骤：**
1. 进入"五险一金"页面
2. 输入工资基数
3. 点击"计算"按钮

**输出结果：**
- 个人缴纳明细（养老、医疗、失业、公积金）
- 单位缴纳明细（养老、医疗、失业、工伤、生育、公积金）
- 个人缴纳总计
- 单位缴纳总计
- 合计金额

**缴费比例说明：**
- 养老保险：个人8%，单位16%
- 医疗保险：个人2%，单位10%
- 失业保险：个人0.5%，单位0.5%
- 工伤保险：个人0%，单位0.5%
- 生育保险：个人0%，单位0.8%
- 住房公积金：个人12%，单位12%

**示例：**
- 工资基数：10,000元
- 个人缴纳：2,250元
- 单位缴纳：3,980元

### 3. 企业成本计算器

**使用步骤：**
1. 进入"企业成本"页面
2. 输入员工月工资
3. 输入员工数量
4. （可选）输入其他月度成本（如办公设备、培训等）
5. 点击"计算"按钮

**输出结果：**
- 月度成本明细（工资总额、五险一金、其他成本）
- 年度成本明细
- 月度总成本和年度总成本

**示例：**
- 员工月工资：10,000元
- 员工数量：5人
- 月度总成本：约69,900元（含五险一金）
- 年度总成本：约838,800元

### 4. 贷款计算器

**使用步骤：**
1. 进入"贷款计算器"页面
2. 输入贷款金额（元）
3. 输入年利率（如4.9表示4.9%）
4. 输入贷款年限
5. 选择还款方式：等额本息或等额本金
6. 点击"计算"按钮
7. （可选）点击"查看还款计划"查看详细还款明细

**还款方式说明：**
- **等额本息**：每月还款金额固定，适合收入稳定的人群
- **等额本金**：每月还款本金固定，利息逐月递减，前期压力大但总利息少

**输出结果：**
- 贷款金额和期限
- 月还款额（等额本息）或首月/末月还款额（等额本金）
- 支付总额
- 支付利息
- 详细还款计划表（可选查看）

**示例：**
- 贷款金额：1,000,000元
- 年利率：4.9%
- 贷款年限：30年
- 等额本息月供：约5,307.27元
- 总利息：约910,616.19元

## 常见问题

**Q: 计算结果准确吗？**
A: 本应用基于2019年最新个税法和通用社保政策计算，结果仅供参考。各地政策可能有差异，具体以当地官方规定为准。

**Q: 五险一金比例为什么和我的不一样？**
A: 不同城市的五险一金缴费比例有所差异，本应用使用的是较为通用的比例。实际缴费请以当地社保部门规定为准。

**Q: 专项附加扣除包括哪些？**
A: 包括子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人等六项。具体金额请参考税务部门规定。

**Q: 等额本息和等额本金哪个更好？**
A: 
- 等额本息：每月还款压力均衡，适合收入稳定的家庭
- 等额本金：总利息较少，适合前期还款能力强的家庭
